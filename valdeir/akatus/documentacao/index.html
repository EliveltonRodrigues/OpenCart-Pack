O que é o projeto.
O projeto "Akatus Transparente", foi criado para facilitar a compra para o usuário final. O projeto quando instalado tem a função de tornar a compra mais fácil pois ele 'elimina' o famoso 'redirecionamento'.
Vantagens
Pagamento feito totalmente em seu e-commerce ou site
O cliente fica no ambiente do seu e-commerce ou site durante todo o processo de compra, sem necessidade de cadastro ou páginas intermediárias de pagamento.

Compra por 1 clique
Na primeira compra, você poderá optar por salvar os dados do cartão de crédito do seu cliente. Com isso, o cliente poderá comprar novamente em seu e-commerce ou site sem precisar digitar todos os dados de pagamento. Esta funcionalidade torna o processo de compra muito mais simples e rápido.

Segurança de dados feita pelo Akatus
Os dados de pagamento dos seus clientes são direcionados diretamente do navegador para o Akatus. Sem passar por seus servidores, assim não precisa se preocupar com a segurança destas informações.

Requisitos.
OpenCart: 1.5.1, 1.5.1.3, 1.5.2.1, 1.5.3, 1.5.3.1, 1.5.4, 1.5.4.1
VQmod: Não
jQuery: 1.7 ou superior
ColorBox: Sim MeioMask: Sim
Instalação
1 - Extraia o arquivo akatus.zip no seu computador.
2 - Copie as pastas "admin", "catalog", "image", "valdeir", "vqmod" e o arquivo "retorno_akatus.php" para a raiz de sua loja.
3 - Acesse http://www.SEUDOMINIO.com.br/valdeir/akatus/.
4 - Clique em Instalar
5 - Faça seu login com o usuário e senha do admin.
6 - Preencha todos os dados.
7 - Vá no painel administrativo.
8 - Acesse o menu Sistema > Usuários > Grupos de Usuário.
9 - Dê permissões para akatus/crypt e payment/akatus
Api Key e Api Nip
Para capturar sua Api Key e Api Nip acesse:
1 - https://www.akatus.com/
2 - Acesse o menu Integração
3 - Acesse a opção Chaves de Segurança
Retorno Automático
Para seu cliente receber a notificação do status de pagamento automáticamente é necessário casdastrar sua url de retorno automático, para isso basta acessar:
1 - https://www.akatus.com/
2 - Ir no menu Integração > Redirecionamentos
3 - Marcar a opção Habilitar redirecionamento?
4 - Cadastar sua url em Cadastre aqui sua URL para notificação de pagamentos instantânea (ex: http://www.SEUDOMINIO.com.br/retorno_akatus.php).
5 - Clicar em Salvar Alterações .

Configuração Básica
Status - Opção para habilitar ou desabilitar o módulo de pagamento.
Api Key - Chave para poder ter liberação do módulo (Como receber o Token).
Api Nip - Chave para poder ter liberação do módulo (Como receber a Key).
Notificar Cliente - Notifica o cliente automáticamente quando o status do pagamento for alterado no site do Akatus.
Desconto Cartão, Débito e Boleto - Define um desconto em reais para cada forma de pagamento (Ps.: O valor deverá ser preenchido com casa decimais. Ex. 12.00).
Aguardando Pagamento - Status quando o pagamento for iniciado.
Aprovado - Status quando o pagamento for aprovado pela Akatus.
Cancelado - Status quando o pedido for cancelado.
Em análise - Status quando o pagamento estiver em análise.
Área Geográfica - Área Geográfica onde o pagamento via Akatus for aceito. (O Akatus só aceita da região brasileira), portanto não se não tiver essa opção, crie.
Ordem - Ordem em que o pagamento via Akatus ira aparecer na forma de pagamento quando o cliente estiver finalizando a compra.
Parcelas
O módulo Akatus Checkout Transparente Pro oferece total flexibilidade e transparência para você configurar as suas regras de parcelamento de acordo com a estratégia e necessidade do seu negócio.
Para aceitar um pagamento parcelado com cartão de crédito você deverá acessar:
1 - http://www.SEUDOMINIO.com.br/admin/
2 - Acessar o menu Extensões > Pagamento > Clique em editar ao lado de Akatus > Logo após isso selecione a opção Parcelas (ao lado esquero)
3 - Clique em Adicionar e define os valores "Para"

Desinstalação.
1 - Acesse: http://www.SEUDOMINIO.com.br/valdeir/akatus/.
2 - Clique em Desinstalar.
3 - Faça seu login.
4 - Confirme que deseja desinstalar o módulo.
5 - Pronto! O Módulo foi desinstalado com sucesso.
